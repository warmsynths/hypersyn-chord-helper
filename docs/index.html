How To

1. Enter a chord progression — e.g. Am7 Dm9 G13 Cmaj7

2. Hit Convert — get hex codes & interval codes ready for Hypersyn on M8.

3. Drag the keyboard — slide up/down on any chord's piano keys to cycle voicings.

4. Toggle Interval Mode — click the button to switch between root-baked hex and interval-only hex (shown in green).

Source progressions from Onemotion, ChordSeqAI, or ChatGPT.

Root-baked: includes the chord root — paste directly into Hypersyn.
Interval: chord shape only — you set the root in the phrase view.
